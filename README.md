# ascii-art-converter

A dark, minimal browser-based image-to-ASCII art converter. No backend, no server, everything runs locally in your browser.

## Features
- **Input:** PNG, JPG/JPEG, WebP
- **Output:** TXT download, PNG export, JPG export, copy to clipboard
- **Character width:** 40 to 500 — experimental high-detail mode at 220+
- **Quick presets:** 80 / 160 / 280 / 320 / 420 / 500
- **Character sets:** Dense, Ultra detail, Blocks, Balanced, Minimal
- **Color modes:** Foreground color, Sampled, Source color (full image color)
- **Invert brightness** mapping toggle
- **Blank background threshold** control
- **Live stats:** width × height in characters, total character count

## Usage
Open `index.html` in any modern browser. Or visit the live Vercel deployment.

## Tech
Pure HTML + CSS + Vanilla JS. No dependencies, no build step.
